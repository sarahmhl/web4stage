# Web4Stage

Application web de gestion de stages.

## Fonctionnalites
- authentification par role
- gestion des offres
- gestion des entreprises
- suivi des candidatures
- wish-list etudiant

## Stack
- PHP
- MySQL
- Apache
- HTML / CSS / JavaScript
- MVC

## Lancement
1. Placer le projet dans `C:\xampp\htdocs\projet web`
2. Demarrer Apache et MySQL
3. Creer la base `web4stage`
4. Importer `database/01_schema.sql`
5. Importer `database/02_seed.sql`
6. Ouvrir `http://localhost/projet%20web/public/index.php/`
